## Usage

Add it to your MDX components.

```tsx
import { Card, Cards } from 'fumadocs-ui/components/card';

<MDX
  components={{
    Card,
    Cards,
  }}
/>;
```

See [Markdown](/docs/ui/markdown#cards) for usages.

### Cards

The container of cards.

### With Icons

```mdx
import { HomeIcon } from 'lucide-react';

<Cards>
  <Card
    icon={<HomeIcon />}
    href="/"
    title="Home"
    description="Go back to home"
  />
</Cards>
```

<Callout title="Tree Shaking" type="warn">

If you're using Contentlayer, or other libraries for rendering MDX, ensure that
tree shaking is working properly.

Most of the icon libraries support importing icons individually.

```tsx
import HomeIcon from 'lucide-react/dist/esm/icons/home';
```

As a workaround, you can pass icons to MDX Components too.

```tsx
import { HomeIcon } from 'lucide-react';

const components = {
  ...defaultComponents,
  HomeIcon,
};
```
